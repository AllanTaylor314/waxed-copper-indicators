Golem statue item overrides for Minecraft 1.21.9 through 1.21.11 (resource pack formats 69–75).

In that window, oxidised golem body textures use names like exposed_copper_golem.png.
From 26.1 onward, vanilla uses copper_golem_exposed.png instead. The base pack targets
26.1+ naming; this overlay patches only the four waxed golem statue item definitions so
they resolve the correct textures on older clients.

Applied automatically via pack.mcmeta — do not enable this folder as a separate pack.
Regenerate with: python generate_items.py
